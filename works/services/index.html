🏛主な公開物🏛️

これまでに公開した主なツール・サービスたちです。

📌 MEGAMOJI(2017)

開く

各種チャットツールでの使用を想定した、カスタム絵文字作成ツールです。画像を用意しなくてもテキストから簡単に絵文字を作れるほか、派手なアニメーションを簡単につけることもできます。Qiita記事の小バズからじわじわユーザーが増え続け、2023/02 時点で MAU 1.7万人を達成。

コア技術: 2Dグラフィック, SVG操作, シェーダ芸, GIFエンコード, Web Workers

📌 HRTFy(2022)

開く

ブラウザ上で立体音響を制作できるツールです。TRPG のオンラインセッションなどで、効果音の演出に使うことを想定しました。手持ちの音声素材ですぐに遊べます。

コア技術: 音声処理 (HRTF), 3Dグラフィック

📌 sky-color-clock.el(2017)

GitHub,技術解説

今の時刻を「空の色」として視覚的に感じることのできる時計ウィジェットです。仕事中でも「空が夕焼けてきたなあ」といった""エモ""を感じられるようにと思い作りました。任意で Openweathermap から現在の天気を取得して、曇り度合いなども反映できます。Emacs 上で動作しますが、有志の方がデスクトップアプリ版,Vim版,VSCode版などの派生版も作ってくださっています。

コア技術: 簡易的な天体計算, Openweathermap API

📌 いい感じの初手率計算機(2021)

開く

カードゲームのデッキ構築支援ツールです。カードの投入枚数が手札にどう影響するかを直感的に把握できます。内部的には、枝刈りしながら DFS で考えられる手札のパターンを列挙、それぞれの場合の数を計算します。

コア技術: 深さ優先探索＋枝刈り

📌 YouTubeサイドプレイヤー(2007)

（現在は削除）

デスクトップの片隅で YouTube 動画を連続再生しておける Windows 用ソフトです。今となっては当たり前の機能という感じですが、当時はまだ YouTube が流行り始めた頃で、 Picture-in-picture も連続再生機能も公式には存在しませんでした。いくつかの雑誌に掲載されました。

外音取込君(2021)

開く

マイクなどの音声入力を、そのままイヤホンなど別のデバイスに流すツールです（いわゆる仮想ケーブル）。イヤホンをしたまま外の音も聞きたい、という場面があったので作ってみました。ノイズ除去や EQ など基本的な音声処理にも対応したので、簡易的なマイクフィルタとしても使えるかもしれません。

コア技術: 音声処理/IO, PWA

受肉実験。(2021)

開く

Web カメラで表情や姿勢を読み取って、MMD や VRoid の 3D モデルを連動させる実験です。「ブラウザだけでお手軽 VTuber」的なものを作りたかったので、あえて動きに制約をつけて、ちょっと 2D っぽく動くようになっています。技術検証ができたところで満足してしまったので、 UI などはちゃんと作られていません。

コア技術: 機械学習, 3Dグラフィック

ネガポジスライサー(2020)

開く

画像を明度でスライスして、複数の単純モノクロ（２値）画像に分解するツールです。基板アートをやりたい時に便利です。

コア技術: 画像処理

背景抜き太郎(2018)

開く

デザイナーさんが爆速でタイムリーなクソコラを上げてくるのが羨ましくて、背景をサクッと抜くツールを自前で書いてみました。マウスでざっくりとしたヒントを与えて「やっていく」ボタンを押すと動きます。実装しやすそうだったので GrowCut アルゴリズムを使いました。

コア技術: 画像処理 (GrowCut), Web Workers

slack_inviter(2018)

GitHub,技術解説

Slack の招待リンクを作るツールです。公式に招待リンク機能が実装されるまで、大学 OB Slack の運営に使っていました。生成した招待リンクは Google Apps Script で管理され、同じリンクは１度しか使えないようになっています。

コア技術: Slack API, Google Apps Script

OLED Designer(2018)

開く

「Helix」系自作キーボード向けのドット絵エディタです。ディスプレイに表示したい画像をポチポチ描くと、その画像を表示するための C プログラムが生成されます。逆に生成されたプログラムを読み込んで編集することもできます。

YouTubeサイドバーメーカー(2007)

（現在は削除）

YouTube の動画をブログの片隅に設置できる、いわゆる「ブログパーツ」ジェネレータです。いくつかの雑誌や、窓の杜NEWSなどに掲載されました。
