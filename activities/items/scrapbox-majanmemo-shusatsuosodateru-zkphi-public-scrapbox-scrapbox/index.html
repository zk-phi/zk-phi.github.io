麻雀メモ

グループを見つける

大前提として、どんな牌でも、好きな牌をあと２枚持ってこれるならメンツになる。

 例： １ は、 ２ と ３ を持ってこれるならメンツになる

逆にいえば、「あと２枚くればいい感じになる」は何も揃ってないのと同じ。

したがって、すべての牌は「A. すでにメンツになってる牌」「B. あと１枚でメンツになれそうな牌」「C. あと２枚こないとメンツになれない牌 (浮き牌)」のどれかに必ず属する。

 例： (一三六九①①②⑤１２３７７) を持ってる場合
  (１２３) は「A. すでにメンツ」
  (一三), (①①), (①②), (７７) は「B. あと１枚でメンツ」
   ※ ① は２つのグループに属している
  六九⑤ は「浮き牌」

麻雀の上がり形は「A を４つ、 B を１つ、の計５ブロック」なので、

 浮き牌を育てて B を作る
 B を育てて A を作る

のように牌を育てていくことで上がりが近づいてくる (シャンテン数が減ってくる) ゲームになっている。

牌の切り方

上の仕組みを考えると、最速でテンパイまでたどり着く (シャンテン数を単調減少させる) にはざっくり以下を考えればいい。たぶん。

 A と B 合わせてすでに５セットあるなら、浮き牌は捨ててよい
 A と B 合わせて６セット以上あるなら、育ちにくそうな B を捨てる
 逆に A と B 合わせて５セットに満たないなら、どこかに B をもう１つ作る必要がある

実際には点を稼ぐために、多少回り道をしてもドラを育てるとか、そういうケースはありそう。あるいは回り道をした方が結果的に早く上がれるケースとかもあるかも？

B グループの分類

B グループは必ず次のどれかの形になる。

ターツ (あと１枚でシュンツ)

 両端のどっちかがくるとシュンツになる（リャンメン待ち）
  例： (２３) → １か４で揃う
  受け入れが一番広い、最強

 片端がくるとシュンツになる（ペンチャン待ち）
  例： (１２) → ３で揃う
  ドンピシャが来ないと揃わないし、タンヤオにもならないし、ツラい

 真ん中が来るとシュンツになる（カンチャン待ち）
  例： (３５) → ４で揃う
  ドンピシャが来ないと揃わないけど、２か６が来るとリャンメンになる (良形変化) のでペンチャンよりマシ

トイツ (あと１枚でコーツ)

 同じのが来るとターツになる（シャンポン待ち）
  例： (２２) → ２で揃う
  ４枚ある２のうち２枚をすでに持ってるところで、さらにもう一枚２を引いてこないといけないので相当厳しい
   ポン、ロンでもらってくるのが現実的
  手札に１〜２セットあるのがちょうどいい
   × ひとつもない場合 → テンパイしたときジャントウを待つ形 (単騎待ち) になってしんどい
   ○ １セットある場合 → ジャントウとして使う
   ○ ２セットある場合
    ２セットあるうちのどちらかをコーツ、余った方をジャントウに使えば良い、強い
    例： (１１２２) → １がきても２がきてもジャントウ＋コーツで上がり
   × ３セットある場合
    うち１セットは自力で揃えないと (あるいはポンしないと) テンパイまでいけないのでしんどい
    一番弱そうなやつを優先的に切りたい

リャンメン待ちになってるターツを「リャンメンターツ」と言ったり、たとえば６ (ローソー) を待っているカンチャンターツ (５７) を「カンローソー」と言ったりする。

基本的に、揃いにくそうな形の B より、揃いやすそうな形の B を手札に残す方が早く上がれる可能性が高い (牌効率)。同様に、弱い B に育ちやすそうな浮き牌 (1, 2, 8, 9, 字牌) より、強い B に育ちやすそうな浮き牌を手札に残す方が早く上がれる可能性は高い。

捨て牌によって話が変わってくることもある。たとえば (２３) のリャンメンでも、１がすでに場に 4 枚捨てられてるなら、４しか期待できないからあんまり強くない。あるいはたとえば露骨に特定のマークを集めてそうな相手がいる場合、そのマークの牌を待っていても捨ててくれないかもしれない。

捨てようと思っていた孤立牌の周りを引いた場合なども、落ち着いてどのブロックが強そうか考える。たとえば (②２４) を持っていて、②を切って (２４) でやっていくつもりだったとしても、もし④を引いてきたら、ちゃんと (②④) と (２４) を比較してからどちらかを切る。具体的には、場に捨てられている③, ３の数などを比べる。

２つの B が１つの牌を共有する形 (複合形)

３枚で１ブロックを待つことになるので物理的にかさばるけど、受け入れは広がる。

リャンカンチャン
 例: (２４６) ... (２４) と (４６) がカンチャン → ３か５がくれば揃う
 どっちも自分は持っていない牌なので、山に各 4 枚づつ、計 8 枚ある。リャンメンほどじゃないけど結構強い

以下はシャンポンが絡むので、リャンカンよりは弱い。

カンチャントイツ
 例： (２２４) ... (２２) がシャンポン、 (２４) がカンチャン → ２か３がくれば揃う

リャンメントイツ
 例： (２２３) ... (２２) がシャンポン、 (２３) がリャンメン → １か２か４がくれば揃う

ペンチャントイツ
 例： (１２２) ... (１２) がペンチャン、 (２２) シャンポン → ２か３がくれば揃う

※リャンメン＋カンチャンは、よく見ると受け入れが広がってない
 例： (２４５) … (４５) リャンメンの時点で３, ６を受け入れているので、２がついても意味ない

弱い方の B を作っている牌 (上の例たちでは２) を、待ちを広げて強化する牌という意味で「フォロー牌」という。

必ず２面以上で待てるので一見強そうだけど、特にシャンポンが絡む形はそこまで強くない。たとえばペンチャントイツ (１１２) は１, ３の２面で待っているけど、１を重複して持っておくことで増えている待ちは１だけ。しかもそれはあと２枚しかない。そもそもペンチャン自体も強いとは言えない形なので、あえて１を切って代わりに適当な孤立牌をキープして、それがリャンメンなどいい形に育つことを期待するというのもあり。

実は完成してるメンツに注意

２つのメンツやジャントウがくちゃっとなってるやつ。パッと見でパターン認識できるようになりたい。

シュンツとシュンツ
 例： (１２３４５６) → (１２３) と (４５６) が隣り合ってる
 例： (１２３３４５) → (１２３) と (３４５) が重なってる
 例： (１２２３３４) → (１２３) と (２３４) が重なってる
 例： (１１２２３３) → (１２３) が２セット (イーペーコー)

シュンツとトイツ
 例： (１２３４４) → (１２３) と (４４) が隣り合ってる
 例： (１２２２３) → (１２３) と (２２) が重なってる

多面待ちを見抜く

「くちゃっとなったメンツ」に育つタイプの B グループ。わかりづらいけどよく出てくる形なので、ぱっと見抜けるようになりたい。全体を２ブロックとして数える。

以下は特に覚えておいた方がいい形。らしい。

階段をふた通りに使えるパターン
 例： (２２３４) ... アリャンメン
  (２２) ＋ (３４) でジャントウとリャンメンがくっついてる
   リャンメンなので強そうだが、２をすでに２枚持ってるのでちょっとしんどい
  ２ ＋ (２３４) とみて、単騎待ちの頭を取り替えることができる
   たとえば８を引いて２を捨てれば、８の単騎待ち
 例： (２３４５) ... ノベタン
  (２３４) + ５ と見ると、５の単騎
  ２ + (３４５) と見ると、２の単騎
  ジャントウを２面で待ててお得、ジャントウがない場合に積極的に残したい形
 例： (２３４５６) ... ５連形
  (２３４) ＋ (５６) と見ると、 (５６) のリャンメン (４, ７待ち)
  (２３) ＋ (４５６) と見ると、 (２３) のリャンメン (１, ４待ち)
  ２メンツ目を３面で待ててお得
 例： (４５５６７) ... ターツ＋１メンツ
  (４５６) + (５７) と見ると、５ (７捨て), ７ (５捨て) がくればジャントウができる
  (４５) + (５６７) と見ると、４ (５捨て), ５ (４捨て) がくればジャントウができる
  ４ + (５５) + (６７) と見ると、(５５) のジャントウと (６７) のリャンメン (５, ８待ち, ４捨て)
  どれが揃っても１メンツ１ジャントウなので、実質ジャントウを４面で待っている形
   必ず１枚余るので物理的なコスパはちょっと悪い
 例： (２４５６６７) ... リャンメンカンチャン
  (４５６７) の４個階段に注目
  (４５６) + (２６７) と見ると、(６７) のリャンメン (５, ８待ち, ２捨て)
  (２４５) + (５６７) と見ると、(２４) のカンチャン (３待ち, ６捨て)
  ２メンツ目を３面で待てる
   必ず１枚余るので物理的なコスパはちょっと悪い

コーツが (トイツ＋１) としても使える形
 例： (２２２３４北北)
  (２２２) + (３４) と見ると、(３４) のリャンメン (２, ５待ち、 (北北) は雀頭)
  (２２) + (２３４) と見ると、 (２２) と (北北) がシャンポン (２, 北待ち)
   ２がシャンポンであることで、受け入れが北まで伸びるのがミソ
  どれがきても必ずジャントウ＋２メンツできるので、(２２２３４) 部分を実質２ブロックとして数えられる

リャンカンにシャンポンが重なった形
 例： (３３５７７) ... サンドイッチ形
  (３５７) のリャンカンに注目
  (３３５) + (７７) と見ると、 (３３５) カンチャントイツ + (７７) ジャントウ (３, ４待ち)
  (３３) + (５７７) と見ると、 (３３) ジャントウ + (５７７) カンチャントイツ (６, ７待ち)
  どれがきても必ずジャントウ＋１メンツできる

一番弱い B を強くする

麻雀は４ブロックを A に育てないと上がれないゲームなので、「いい感じの B」よりも「ヤバい感じの B 」が律速要因になりやすい。

 例： 手札がパンパンで (２２３５５８) から１枚切らないといけない場面
  より強い形の (２２３) から１枚切って、より弱い形の (５５８) を守ってあげた方がトータル揃いやすい
  ２を切る → (２３) のリャンメンが残るので十分いけそう
  ５を切る → (５８) のカンチャンが残ってキツそう

4 グループしかない場合

手札に４グループしかない場合、どこかに１グループ生やす必要がある。

一番シンプルなのは孤立牌の周囲±２の牌を引くこと。

 例： ３ → １か５を引くとカンチャン、２か４を引くとリャンメン、３を引くとシャンポン

だけど、「多面待ち」と同様に、もっと５グループ目が生えやすい形もある。

４階段 (ノベタン)
 例： (３４５６)
  (３４５) + ６と考えると、４, ５, ６, ７, ８を引けば５グループ目が生える
  ３ + (４５６) と考えると、１, ２, ３ を引いても５グループ目が生える
  だいたい何がきても５グループ目が生える
  ジャントウを生やしたいときにも便利 (３, ６のどちらがきてもジャントウができる)

中ぶくれ３階段
 例： (３４４５)
  (３４) + (４５) と考えると、すでにリャンメンが２グループあるとも考えられる (２, ３, ５, ６ 待ち)
  (３４５) + ４ と考えると、４を引いても一応５グループ目が生える
  リャンメンができる可能性が高いので長い目て見て強い
  ノベタンと違ってジャントウを生やすのには使えない

階段＋カンチャン
 例： (３４５７)
  (３４) + (５７) と考えると、すでにリャンメンとカンチャンが１ブロックずつあるとも考えられる (２, ５, ６待ち)
  (３４５) + ７ と考えると、７, ８, ９を引いても７にくっつけて５グループ目として使うことができる

４ブロックしかないときにこういう形を見かけたら、孤立牌より優先して温存する。

浮き牌をあえて残す場合

すでに５ブロック揃っている場合、浮き牌はすべて捨ててしまってもテンパイできるが、あえて残すと強いケースもある：

 例： (①②３４５６)
  (３４５) でシュンツが完成しているので、６は切っても上がれる
  あえて残しておくと、もし３±２, ６±２を引いてリャンメンなどができた場合、 (①②) のペンチャンを捨てられる
