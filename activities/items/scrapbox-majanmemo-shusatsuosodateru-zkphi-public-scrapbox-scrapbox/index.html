麻雀メモ

グループを見つける

大前提として、どんな牌でも、好きな牌をあと２枚持ってこれるなら３枚ブロックになる。

 例： １ は、 ２ と ３ を持ってこれるなら３枚ブロックになる

逆にいえば、「あと２枚くればいい感じになる」は何も揃ってないのと同じ。

したがって、すべての牌は「A. すでに３枚ブロックになってる牌」「B. あと１枚で３枚ブロックになれそうな牌」「C. あと２枚こないと３枚ブロックになれない牌 (浮き牌)」のどれかに必ず属する。

 例： (一三六九①①②⑤１２３７７) を持ってる場合
  (１２３) は「A. すでに３枚ブロック」
  (一三), (①①), (①②), (７７) は「B. あと１枚で３枚ブロック」
   ※ ① は２つのグループに属している
  六九⑤ は「浮き牌」

麻雀の上がり形は「A を４つ、 B を１つ、の計５ブロック」なので、

 浮き牌を育てて B を作る
 B を育てて A を作る

のように牌を育てていくことで上がりが近づいてくる (シャンテン数が減ってくる) ゲームになっている。

牌の切り方

上の仕組みを考えると、最速でテンパイまでたどり着く (シャンテン数を単調減少させる) にはざっくり以下を考えればいい。たぶん。

 A と B 合わせてすでに５セットあるなら、浮き牌は捨てていく
 A と B 合わせて６セット以上あるなら、育ちにくそうな B から捨てていく
 逆に A と B 合わせて５セットに満たないなら、どこかに B をもう１つ作る必要がある

実際には点を稼ぐために、多少回り道をしてもドラを育てるとか、そういうケースはありそう。あるいは回り道をした方が結果的に早く上がれるケースとかもあるかも？

B グループの分類

B グループは必ず次のどれかの形になる。

両端のどっちかがくると階段になる（リャンメン待ち）
 例： (２３) → １か４で揃う
 受け入れが一番広い、最強

片端がくると階段になる（ペンチャン待ち）
 例： (１２) → ３で揃う
 ドンピシャが来ないと揃わないし、タンヤオにもならないし、ツラい

真ん中が来ると階段になる（カンチャン待ち）
 例： (３５) → ４で揃う
 ドンピシャが来ないと揃わないけど、２か６が来るとリャンメンになる (良形変化) のでペンチャンよりマシ

同じのが来るとスリーカードになる（シャンポン待ち）
 例： (２２) → ２で揃う
 ４枚ある２のうち２枚をすでに持ってるところで、さらに２をもう一枚引いてこないといけないので相当厳しい
  ポン、ロンでもらってくるのが現実的
 手札に１〜２セットあるのがちょうどいい
  ひとつもない場合 → テンパイしたときジャントウを待つ形 (単騎) になってしんどい
  １セットある場合 → そのままジャントウとして使える
  ２セットある場合
   ２セットあるうちのどちらかをスリーカード、余った方をジャントウに使えば良い
    例： (１１２２) → １がきても２がきても２枚ペア＋スリーカードで上がり
  ３セットある場合
   うち１セットは自力で揃えないと (あるいはポンしないと) テンパイまでいけないのでしんどい

基本的に、揃いにくそうな形の B より、揃いやすそうな形の B を手札に残す方が早く上がれる可能性が高い (牌効率)。ただし、待ってる牌がすでに場に捨てられてる場合などは別。たとえば (２３) のリャンメンでも、１がすでに場に 4 枚捨てられてるなら、４しか期待できないからあんまり強くない。

同様に、弱い B に育ちやすそうな浮き牌 (1, 2, 8, 9, 字牌) より、強い B に育ちやすそうな浮き牌を手札に残す方が早く上がれる可能性は高い。

２つの B が１つの牌を共有する形

３枚で１ブロックを待つことになるので物理的にかさばるけど、受け入れは広がる。

カンチャン＋カンチャン（リャンカン）
 例: (２４６) ... (２４) と (４６) がカンチャン → ３か５がくれば揃う
 どっちも自分は持っていない牌なので、山に各 4 枚づつ、計 8 枚ある。嬉しい。

以下はシャンポンが絡むので、リャンカンよりはちょっと弱い。

カンチャンの一端がシャンポン
 例： (２２４) ... (２２) がシャンポン、 (２４) がカンチャン → ２か３がくれば揃う

リャンメンの一端がシャンポン
 例： (２２３) ... (２３) がリャンメン、 (２２) がシャンポン → １か２か４がくれば揃う

弱い方の B を作っている牌 (上の例では２) を、待ちを広げて強化する牌という意味で「フォロー牌」という。

※リャンメン＋カンチャンは、よく見ると受け入れが広がってない
 例： (２３５) … (２３) リャンメンの時点で１, ４を受け入れているので、５がついても意味ない (捨てられる)

見づらい A グループに注意

２つの A グループ (あるいは A グループと B グループ) がくちゃっとなってるやつ。パッと見でパターン認識できるようになりたい。

階段と階段
 例： (１２３４５６) → (１２３) と (４５６) が隣り合ってる
 例： (１２３３４５) → (１２３) と (３４５) が重なってる
 例： (１２２３３４) → (１２３) と (２３４) が重なってる
 例： (１１２２３３) → (１２３) が２セット (イーペーコー)

階段とペア
 例： (１２３４４) → (１２３) と (４４) が隣り合ってる
 例： (１２２２３) → (１２３) と (２２) が重なってる

多面待ちを見抜く

「見づらい A グループ」に育つタイプの B グループ。わかりづらいけど受け入れが広くて強い形なので、見抜けるようになりたい。全体を２ブロックとして数える。

以下は特に覚えておいた方がいい形。らしい。

４個階段・５個階段をふた通りに使える形
 例： (２３４５) ... ノベタン
  (２３４) + ５ と見ると、５の単騎
  ２ + (３４５) と見ると、２の単騎
  どっちが揃っても１メンツ１ジャントウ
  ※ ２メンツ欲しい場合は (２３) + (４５) のリャンメン２つと見ても ok
 例： (２３４５６)
  (２３４) ＋ (５６) と見ると、 (５６) のリャンメン (４, ７待ち)
  (２３) ＋ (４５６) と見ると、 (２３) のリャンメン (１, ４待ち)
  どっちが揃っても２メンツ
 例： (４５５６７) ... ターツ＋１メンツ
  (４５６７) の４個階段に注目
  (４５６) + (５７) と見ると、５, ７がくればジャントウができる
  (４５) + (５６７) と見ると、４, ５がくればジャントウができる
  ４ + (５５) + (６７) と見ると、(５５) のジャントウと (６７) のリャンメン (５, ８待ち, ４は捨てる)
  どれが揃っても１メンツ１ジャントウ
  ※ ２メンツ欲しい場合は (５７) をカンチャンと見たり (４５) をリャンメンと見ても ok
 例： (２４５６６７) ... リャンメンカンチャン
  (４５６７) の４個階段に注目
  (４５６) + (２６７) と見ると、(６７) のリャンメン (５, ８待ち, ２は捨てる)
  (２４５) + (５６７) と見ると、(２４) のカンチャン (３待ち, ６は捨てる)
  どっちが揃っても２メンツ

スリーカードが (ペア＋１) としても使える形
 例： (２２２３４北北)
  (北北) のシャンポンは孤立しているので置いておいて、 (２２２３４) に注目
  (２２２) + (３４) と見ると、(３４) のリャンメン (２, ５待ち、 (北北) は雀頭)
  (２２) + (２３４) と見ると、 (２２) と (北北) がシャンポン (２, 北待ち)
   ２がシャンポンであることで、受け入れが北まで伸びるのがミソ
  どれがきても必ずジャントウ＋２メンツできるので、(２２２３４) 部分を２ブロックとして数える

リャンカンにシャンポンが重なった形
 例： (３３５７７) ... サンドイッチ形
  (３５７) のリャンカンに注目
  (３３５) + (７７) と見ると、 (３３５) カンチャン/シャンポン + (７７) ジャントウ (３, ４待ち)
  (３３) + (５７７) と見ると、 (３３) ジャントウ + (５７７) カンチャン/シャンポン (６, ７待ち)
  どれがきても必ずジャントウ＋１メンツできる

一番弱い B を強くする

麻雀は４ブロックを A に育てないと上がれないゲームなので、「いい感じの B」よりも「ヤバい感じの B 」が律速要因になりやすい。

 例： 手札がパンパンで (２２３５５８) から１枚切らないといけない場面
  より強い形の (２２３) から１枚切って、より弱い形の (５５８) を守ってあげた方がトータル揃いやすい
  ２を切る → (２３) のリャンメンが残るので十分いけそう
  ５を切る → (５８) のカンチャンが残ってキツそう

4 グループしかない場合

手札に４グループしかない場合、どこかに１グループ生やす必要がある。

一番シンプルなのは孤立牌の周囲±２の牌を引くこと。

 例： ３ → １か５を引くとカンチャン、２か４を引くとリャンメン、３を引くとシャンポン

だけど、「多面待ち」と同様に、もっと５グループ目が生えやすい形もある (くっつき１シャンテン)。

４階段 (ノベタン)
 例： (３４５６)
  (３４５) + ６と考えると、６±２を引けば５グループ目が生える
  ３ + (４５６) と考えると、３±２を引けば５グループ目が生える
  ジャントウを生やしたいときにも便利 (３, ６のどちらがきてもジャントウができる)

中ぶくれ３階段
 例： (３４４５)
  (３４) + (４５) と考えると、すでにリャンメンが２ブロックあるとも考えられる (２, ３, ５, ６ 待ち)
  (３４５) + ４ と考えると、４±２を引けば５グループ目が生える
  リャンメンが残りやすいので強い

階段＋カンチャン
 例： (３４５７)
  (３４) + (５７) と考えると、すでにリャンメンとカンチャンが１ブロックずつあるとも考えられる
  (３４５) + ７ と考えると、７±２を引けばそれを５グループ目として使うこともできる

４ブロックしかないときにこういう形を見かけたら、孤立牌より優先して温存する。

浮き牌をあえて残す場合

すでに５ブロック揃っている場合、浮き牌はすべて捨ててしまってもテンパイできるが、あえて残すと強いケースもある：

 例： (①②３４５６)
  (３４５) で A グループが完成しているので、６は切っても上がれる
  残しておくと、もし３±２, ６±２を引いてリャンメンなどができた場合、 (①②) のペンチャンを捨てられる
