#麻雀基礎まとめ

門前役

人から牌をもらわずにテンパイした場合だけつくやつ。リーチの上位互換。どっちもよく出てくる。

ピンフ
 役牌を使わず、メンツをすべてシュンツにして、リャンメン待ちで上がる
 単騎・ペンチャンで上がるとつかないので注意

イーペーコー
 まったく同じシュンツを２セット作る
 例： (１１２２３３) など
 イーペーコーを２セット作ると３飜 (リャンペーコー)

鳴いてもいい役

出現率順。下に行くほどノーチャンス。もっと狙いやすい役は麻雀メモ / 基本ルールの方に。

サンショクドウジュン (２飜/鳴き１飜)
 同じ並びのシュンツを３種類のマークで作る
 例： (一二三①②③１２３) など

イッツー (２飜/鳴き１飜)
 同じマークの１〜９を全部揃える（123・456・789 の３シュンツを作る）
 人からもらっていいとはいえ、左隣からしかもらえないから狙ってやるのはむずそう

チャンタ (２飜/鳴き１飜)
 すべてのブロックに１・９・字牌を絡める
  ＝ (123), (789), (111), (999), 字牌 が使える
 階段を使わずに作ると、鳴いても２飜 (ホンロートー)
 字牌を使わずに作ると３飜 (純チャン)
 階段も字牌も使わずに作ると役満 (チンロートー)

この辺からかなり厳しい。

サンアンコ (２飜)
 コーツを自力で３つ作る
  ロンで３つ目を作るのもダメ、ツモなら ok
 ロンで４つ目ができると役満 (スーアンコ)
 自力で４つ作る (＝単騎待ちになる) とダブル役満 (スーアンコ単騎)

ショウサンゲン (２飜)
 白發中で２＋３＋３の組を作る (どれを２にしても ok)
 ３＋３＋３を作ると役満 (ダイサンゲン)
 東南西北で２＋３＋３＋３を作ると役満 (ショウスーシー)
 東南西北で３＋３＋３＋３を作るとダブル役満 (ダイスーシー)

サンショクドウコク (２飜)
 同じ番号のコーツを３種類のマークで作る
 例： (一一一①①①１１１) など

国士無双 (役満)
 １・９・字牌を各１枚ずつ (どれか一種類だけ２枚) 集める
 役満の中では出やすい、らしい
 全種ちょうど１枚ずつ集めた上で、最後の１枚を１３面待ちで上がった場合ダブル役満

打点を伸ばすか速く上がるか

多少遠回りになってでも強い役を狙ったほうがいいか。

そもそも上がれなければ強い役を作っても意味ないので、一般にはスピード重視の方が強い。ただし、すでに相手に大差をつけられていて強い役を作らないと勝ち目がない場合などは、高打点を意識する。

あるいは、そこまでスピードを落とさずに打点を伸ばせるなら打点を意識する (ドラが含まれるように調整するとか、１・９が入らないように調整するとか)。特に４飜未満の場合、１飜上がるだけで打点が倍になるので狙うメリットが大きい (上がれる確率が半分になっても、打点が倍なら期待値トントン)。

鳴くかどうか

ポン・チーをするメリットは
 役を速く揃えられる

デメリットは
 手札が減るので、オリると決めた時に捨て札の選択肢が少ない (防御力下がる)
 打点が下がる (リーチやピンフがつかない) 、他に役がないとそもそも上がれない

なので、デメリットが小さい場合、メリットが大きい場合は鳴いていい。

たとえば手札にカンチャン・ペンチャンのような揃いにくそうなブロックがある場合、鳴くことで一気に上がりが近づくのでメリットが大きい。点は下がるが、相手に上がられるよりはマシ。一方、リャンメンやそれ以上のブロックは自力でも揃えられる可能性が高いので、メリットは小さくになる。

逆にとても上がれそうにない手札の場合、オリる可能性が高いので、防御力が下がるデメリットを重くみて鳴かないのが吉。

あるいは鳴けば強い役で上がれそうな場合、満貫以降の打点は上昇幅が小さいので、打点が下がるデメリットより早く上がれるメリットが大きくなる。一方、４飜未満になりそうな場合、１飜で打点が倍違うので、打点が下がるデメリットが大きくなる。

おまけ：運ゲー役

なんか知らんうちについてるやつ。覚えなくていい。

 メンゼンツモ ... ポン・チー・カンせずにツモ上がり
 一発 ... リーチ後１巡以内に上がり
 ハイテイラオユエ ... ラストワン賞
 ハイテイラオユイ ... ラストロン賞
 ダブリー ... 初手でリーチ
 チャンカン ... 相手がカンした牌が待ちだった場合に、横取りしてロン上がりできる
 テンホー ... 親が初手で上がり
 チーホー ... 親以外が初手でツモ上がり
 レンホー ... 親以外が初手でロン上がり

おまけ：ほぼ無理そうな役

たぶん無理。覚えなくてよさそう。

ツーイーソー (役満)
 字牌だけで上がる

サンカンツ (２飜)
 カンを３回やる
 なぜ２飜なのか
 ４回やると役満 (スーカンツ)

リューイーソー (役満)
 ホンイツのヤバい版
 ２, ３, ４, ６, ８ と發だけで上がる
  １, ５, ７, ９には黒や赤が使われてるので「緑一色」じゃない
 シュンツは (２３４) 確定、残りは６, ８, 發のスリーカード

チューレンポートー (役満)
 すべて同じマークで、 １・９を各３枚以上と、２〜８を各１枚以上
  １枚足りないので、１・９のどちらかが４枚になるか、２〜８のどれかが２枚になる
 １・９をちょうど各３枚、２〜８をちょうど各１枚集めた上で、最後の１枚を９面待ちで上がった場合ダブル役満
 レアすぎて上がると死ぬらしい
