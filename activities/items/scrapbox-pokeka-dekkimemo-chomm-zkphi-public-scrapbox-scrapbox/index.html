Scrapbox (2021/1/31)See original

コンセプト

 身内で遊んでる三世代戦 (ABC, BCD, CDE のどれかからレギュレーションを選ぶ) の BCD デッキ
 後１ホラーハウス → 後２マリィナイトウォッチャーをなるべく安定させて、相手の初動を遅らせる
  全力の打ち合いになると、 HP 270 二体で負けになるこちらは不利
 後１の要求値：
  イカが最低１体、ミュウミュウバトル場で１エネ、トラッシュにゲンミミ
 後２の要求値：
  イカをカラマネロに進化、トラッシュにオロヨノ、ミュウミュウ下げてサイコリチャージ＋手張り → また前に出す
  カラマネロが２体立ったら、新品のミュウミュウにサイコリチャージ２回＋手張りもあり

メイン

 ミュウミュウ 2 - オロヨノ 2 - ゲンミミ 2
  デッキコンセプトなのでサイド落ち考慮で 2 枚づつ
  ゲンミミはスタートするとかなりきつい (オルター込みブレイブキャリバーでワンパン etc) のでしんどさはある
   後1ホラーハウスは諦めて、2エネホラーハウス→ポルターガイストで最低限の仕事をしてもらいつつ後ろでMM育てたり
  オロヨノスタートは実質メインアタッカーなのでまあアリ
   ホラーハウス撃たなかった場合はGX技をどこで打つか考えながら回す
    MM、オロヨノのGX技もどちらもだいぶ強い
   MMサイド落ちの場合もオロヨノで戦う

 マーイーカ 3 - カラマネロ 4
  後攻を取るので１ターン目からサポート権がある → 3 枚入ってれば１〜２体立てられる読み
  逆にカラマネロは博士やデデンネで落としやすくするためにフル投入
  スタートしたいポケモンではあるので、 4 入れてもいい気はする

 ミストレ 4 - クイボ 4 - プレシャス 3
  ミストレクイボはクソ強いので脳死フル投入
  プレシャスかタッグコールかは諸説だけど、これ一枚でデデンネにアクセスできるのが大きいのでプレシャス
  プレシャスは 2 だとギリって感じだったので、安定を取って 3

勝ち筋補助

 ソルルナ 1
  わざマシン
  おまもりなしザシアンVをワンパン
  ナイトウォッチャーと合わせると380点でおまもりついたVMAXをいい感じに倒せる
   ナイトウォッチャーから入るとマオスイされにくい

 アーゴヨンGX 1
  わざマシン
  デデンネGXをワンパンできるベンチ狙撃

 ベトベトンアローラベトベトン 0
  わざマシン
  ミュウミュウがジュナイパー・チルタリス・タチフサと戦えるようになる
   ギラティナだとタチフサは見れない
  「技のダメージできぜつ」にならないので、しあわせ卵対策にもなる
  ギラティナとどちらか１枚あれば良さそうな枠

 ギラティナ 1
  GX メタ、特性メタなどに当たったときのサブプラン
  10 点足りないときにやぶれたとびらでダメカン載せられるのも強い
   クロバットVに10点乗せるとベノムシュート圏内になったりする
  アロベトと比べたデメリット
   動かすにはエネを張り直す必要がある、イカが２体は立ってないと厳しい
   タチフサは詰み
   ベンチの枠を食う

 ラティオスGX 0
  前に出して GX デッキを詰ませるサブプラン
  後１クリアビジョンでルカメタ、三神のバリューを落とすことができる
  候補ではあるけどメタ範囲が狭いのでとりあえず 0

 クラハン 4
  相手の動きを遅らせる補助
   アルティメットレイからエネ加速とかされると足止めどころじゃないので
  オロヨノスタートしたとき、これで後ろのエネを剥がしながらオロヨノの GX 技打つと相手のエネが枯れて楽しい

 おまもり 0 - かばん 0
  ミュウミュウの確定数をずらせることがある
  フルパワーのドレッドエンドでワンパンされない
  相手がそこまでぶん回ってる次点で負けという感じがするので、いったん 0

リソース

 博士 4 - マリィ 4
  デデンネへのアクセス・デッキ圧縮が豊富なのでこれくらいでも触れる感じ
  マリィにはナイトウォッチャーと合わせて相手の手札を２枚にする役割もある

 デデンネGX 2
  １〜２ターン目にがっつりぶん回したいコンボデッキなので多め
  トラッシュしたいカードが多い＆プレシャスで持ってこれるのでクロバットよりはデデンネ
  なんなら3欲しい

干渉系

 ボス 0
  ベンチ狙撃もあるし、コンボの安定を最優先にしていったん 0
  ギラティナ抜くなら必要かも (ベノムシュートでクロバットVを取れない)

 リセスタ 0
  中盤でマリィナイトウォッチャーの強い版みたいな動きができる
   マリィは4枚しかないのでそんなに都合よく決まらない

入れ換え

 入れ換え 4 - ふうせん 3
  サイコリチャージがベンチにしか加速できないので、コンボを安定させるために多め
  ふうせんはできればミュウミュウよりカラマネロに張りたい
  後半余りがちだけどこれ以上は削れない感じがする

エネ

 トキワ 3
  GX をトラッシュしつつエネを持ってきたり、エネをトラッシュしてサイコリチャージしたりできる
  クラハンとコンフリクトしてる感じはあるけど、あるとないで安定感がだいぶ違う
  ２にして基本超増やす選択はありそう

 ホラー超 4 - 基本超 5
  安定して動けるスレスレの枚数、ホラー超を減らして基本超を増やすのも選択
  トキワ抜くならもう少し欲しい
  ホラー超があるとナイトウォッチャーだけで 320 ~ 340 くらいまで見れるようになる
