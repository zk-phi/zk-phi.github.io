麻雀メモ

-------- 基本ルール

出てくる牌

 各４枚
 種類
  1~9 のマンズ（満, m）、ピンズ（丸, p）、ソーズ（棒, s）
   Mが２個並んだやつはソーズの８（棒で遊ぶな？？）、鳥はソーズの１（なんで？？）
   各マークの「５」だけ、４枚中１枚、「赤でプリントされたレアカード」がある
  字牌
   東西南北
   ハク（無地）、発、中

上がる条件

 ３枚ブロックを４つと、２枚ブロックを１つの、「計５ブロック」をそろえる
  ２枚ブロックは同じカードのペアのみ
  ３枚ブロックはスリーカードまたは階段
  「計５ブロック」は重要なのでいつも意識する
 ただし、役が最低一つ成立していないとダメ
 誰が一人が上がったら他の人は上がれない

ドラ

 左上に「誰もドローできないカード」が裏で５枚置いてある
  これがあるので、牌によってはそもそも同じカードが４枚存在しない場合もある
 ゲーム開始時に一枚表にする
 表になっているカードの次のカード（例、９なら次は１）を持って上がるとボーナスがもらえる
 「赤でプリントされたレアカード」を持って上がってもボーナスがもらえる

できるアクション

 山から一枚引いて、一枚捨てる
 左の人が捨てたカードをもらって、階段を作る（チー）
 だれかが捨てたカードをもらって、スリーカードを作る（ポン）
 だれかが捨てたカードをもらって、上がる（ロン）
 カン
  1. 手札に自力でフォーカードを作った時に、それを公開して３枚ブロック一個分として扱う
   ４枚を３枚として扱うことになるので、代わりに一枚ドローできる
  2. だれかが捨てたカードをもらってフォーカードを作って、３枚ブロック一個分として扱う
  どちらかをやると、裏になっているドラが一枚開いて、ドラが増える
   ボーナスが増えるので、上がれそうなら狙ってもいいけど、上がれなさそうならそっとしとく

上がるために覚えたい役

 役を最低１つは作らないとそもそも上がれないので、その条件を満たすために覚えたい役
  他の役は、↓の役の上位互換みたいなやつが多いので、「ボーナス点」と思っておけばよさそう
 リーチ
  「誰からもカードをもらわずに」あと１枚のところまで揃えて、リーチを宣言する
   最後の一枚はロンで揃えても ok
   一度リーチを宣言したら、それ以降に引いた牌は手札と入れ替えられない
   とりあえずこれを狙おう
    とりあえずリーチって言っとけばチョンボにはならない
 誰かからカードをもらわざるを得ないときに使える役（下に行くほど優先度低）
  役牌
   字牌でスリーカードを一つ作る
    ただし方角のカードは、自分が座ってる席の方角か、全体で共通の方角しか使えない
     たとえば東１局で自分が西の席に座ってたら、南と北は役なし
     東１局で自分が東の席に座ってる時、東でスリーカードを作ると効果が二倍
    ２枚ペアを素引きしてる場合、ポンで揃う可能性がわりとある
     逆に単品で引いちゃった場合は、早々に諦めて捨てるのも手
  タンヤオ
   ２〜８のカードだけで上がる
    字牌も使っちゃダメ
    わりと作りやすいので、人からカードもらわないと上がれなさそうなら積極的に１・９捨てる
  トイトイ
   階段を使わずに上がる
    スリーカード４つと、ペア１つ
    ポンし放題なので意外と狙える、初手でペアがいっぱいあるときはワンチャン
  ホンイツ
   一種類のマークだけで上がる、ただし字牌は使ってもいい
   字牌も使わずに上がれた場合はさらにボーナスがつく（チンイツ）
   初手で丸も棒も１枚ずつしかない、みたいなときはワンチャン
  イッツー
   同じマークの１〜９を全部揃える（123・456・789 の３ブロック作る）
    人からもらっていいとはいえ、左隣からしかもらえないから結構むずい
 特殊な上がり方
  チートイツ
   ２枚ペアを７ブロック作る
   そもそもポンもチーもできないので、６ブロックまでは自力で作る必要がある
   ２枚ペアがすでに４〜５ブロックあります、みたいなときはワンチャン

------- 手札の揃え方

大前提として、どんなカードでも、好きなカードをあと２枚持ってこれれば３枚ブロックになる

 例：１は、２と３を引いてくれば３枚ブロックになる
 逆にいえば、「あと２枚くればいい感じになる」は何も揃ってないのと同じ

「A. すでに３枚ブロックになってる牌」「B. あと１枚で３枚ブロックになれそうな牌」に注目する

 例：あるマークの 1, 2, 3, 7, 8 を持ってる場合
  1, 2, 3 は「A. すでに３枚ブロックになってる」
  7, 8 は「B. あと１枚で３枚ブロックになれそう」
 麻雀は B を育てて A にしていく、というゲーム
  ただし A と B 合わせてそもそも５セットない場合は、 B から作る必要がある
  逆に A と B 合わせてすでに５セットある場合、６セット目以降はいずれ切ることになる
   手札が必要なものでパンパンになってきたら、揃いにくそうなものから切っていく
 A にも B にもなっていないカードは、基本的に無価値
  「あと２枚くればいい感じになる、は何も揃ってないのと同じ」なので、そのカードである必要性はない
   ＝ 他の牌に交換してもなんら問題ない、切っていい
  ただしドラとか役牌なら別

B は次のどれかの形になる

 両端のどっちかがくると階段になる … 例：2, 3 → 1 か 4 で揃う（リャンメン）
  最強（受け入れが広い）
 片端がくると階段になる … 1, 2 → 3 で揃う（ペンチャン）
  ドンピシャが来ないと揃わないし、タンヤオにもならないし、弱い
 真ん中が来ると階段になる … 例：3, 5 → 4 で揃う（カンチャン）
  ドンピシャが来ないと揃わないけど、 2, 6 が来るとリャンメンに持ち変えられるので悪くない
 同じのが来るとスリーカードになる … 例：2, 2 → 2 で揃う（シャンポン）
  ドンピシャが来ないと揃わないけど、見た目より受け入れが広い
  なぜ？麻雀はどのみち２枚ペアが一つ必要なので、２つあったらどちらかがスリーカードになれば良い
   例： 1, 1, 2, 2 → 1 がきても 2 がきても２枚ペア＋スリーカード
  このメリットは３つ目以降の２枚ペアには働かない
   なぜ？麻雀であがるのに必要な２枚ペアは１つだけだから
  最低一つはないと、最終的に単騎待ち (２枚ペアの相方を待つ) になってつらい
 ２つ以上間が空いている牌は互いに無関係
  「あと２枚くればいい感じになる、は何も揃ってないのと同じ」なので
  例: 1, 4 → 2, 3 がくれば 1, 2, 3 や 2, 3, 4 ができるけど、そもそも 1 単体でも 2, 3 がくれば 1, 2, 3 になる

実は揃ってるパターンを見抜く

 例：階段と階段
  例： 1, 1, 2, 2, 3, 3 → 1, 2, 3 が２セット
  例： 1, 2, 2, 3, 3, 4 → 1, 2, 3 と 2, 3, 4 が重なってる
  例： 1, 2, 3, 3, 4, 5 → 1, 2, 3 と 3, 4, 5 が重なってる
 例：階段とペア
  例： 1, 2, 2, 2, 3 → 1, 2, 3 と 2, 2 が重なってる
  例： 1, 2, 3, 4, 4 → 1, 2, 3 と 4, 4 が隣り合ってる
 例：階段に挟まるペア・その逆
  例： 1, 2, 3, 4, 4, 5, 6, 7 → 1, 2, 3 と 5, 6, 7 に 4, 4 が挟まってる
  例： 1, 1, 2, 3, 4, 5, 5 → 1, 1 と 5, 5 に 2, 3, 4 が挟まってる

複数の B が一端 (両端) を共有しているパターンを見抜く

 ３枚で１ブロックを待つのでかさばるけど、受け入れは広がる
 カンチャン＋カンチャン＝リャンカン (2, 4, 6 → 2, 4 と 4, 6 がカンチャン … 3, 5 待ち)
  3 も 5 も自分は持っていない牌なので、山に各 4 枚づつある → 揃いやすい
 カンチャンの一端がシャンポン (2, 2, 4 → 2, 2 がシャンポン、 2, 4 がカンチャン … 2, 3 待ち)
 リャンメンの一端がシャンポン (2, 2, 3 → 2, 3 がリャンメン、2, 2 がシャンポン … 1, 2, 4 待ち)
 リャンメン＋カンチャンは、よく見ると受け入れ広がってないので無駄
  例： 2, 3, 5 … 2, 3 リャンメンの時点で 1, 4 を受け入れているので、 5 がついても意味ない

多面待ちを見抜く

 「複数の見方」ができるカタマリは受け入れが広い
 ４階段・５階段をどこで分けるか
  例： 2, 3, 4, 5, 6
   2, 3, 4 ＋ 5, 6 と見ると、 5, 6 のリャンメン (4, 7 待ち)
   2, 3 ＋ 4, 5, 6 と見ると、 2, 3 のリャンメン (1, 4 待ち)
  例： 2, 4, 5, 6, 6, 7 (リャンメンカンチャン)
   4, 5, 6 ＋ 7 と見ると、 6, 7 のリャンメン (5, 8 待ち, 2 は捨て)
   4 ＋ 5, 6, 7 と見ると、 2, 4 のカンチャン (3 待ち, 6 は捨て)
 スリーカードをペア＋１と見てみる
  例： 2, 2, 2, 3, 4, 9, 9
   2, 2, 2 + 3, 4 と見ると、 3, 4 のリャンメン (2, 5 待ち、 9, 9 は雀頭)
   2, 2 + 2, 3, 4 と見ると、 2, 2 / 9, 9 のシャンポン (2, 9 待ち)
 リャンカンをどこで分けるか
  例： 3, 3, 5, 7, 7
   3, 5 + 7 と見ると、 3, 5 カンチャン ＋ 7, 7 シャンポン (4, 7 待ち, 3 は捨て)
   3 + 5, 7 と見ると、 3, 3 シャンポン ＋ 5, 7 カンチャン (3, 6 待ち, 7 は捨て)

 目当てのカードがあと何枚眠っているかを考える
  受け入れが広くとも、該当する牌がすでに捨てられていたら意味ない
  たとえば自分が 2 を待っているとき、場に 2 がすでに 3 枚捨てられているなら、残る 2 は 1 枚しかない
