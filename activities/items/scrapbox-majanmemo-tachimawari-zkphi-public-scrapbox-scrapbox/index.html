麻雀メモ

オリ

自分が上がれなさそうな場合、相手がヤバい役を作ってそうな場合は、自分が上がることよりも放銃しないことを優先して、手が悪くなってでもロンされにくい牌 (安全牌) を捨てる (オリ)。

ヤバそうな役は相手の動きでなんとなくわかる場合もある。

 特定の色以外をどんどん捨てる → ホンイツ系…？
 使いやすい牌 (２〜８)、強い牌 (ドラ) をいきなり捨てる → チャンタ系…？国士無双…？
 どんどんポンする → トイトイ…？
 同じ色をどんどんチーする → イッツー…？

現物

ルールで「捨てたことのある牌はロンできない」(フリテン) と決まってるので、相手がすでに捨ててる牌は捨てても「絶対」ロンされない。 100% 安全牌なので、オリると決めたら真っ先に捨てる。

リーチ以降に誰かが捨ててスルーされた牌も同様に 100% 安全牌。

捨てる牌が何に当たるか考える

現物を使い切った場合。捨てる牌の危険度を、「どんな待ちに当たりそうか」によって判断する。

何も情報がない場合、通常、上から下の順で安全。特に字牌はシャンポンにしか当たらないので、安全 (字牌のシャンポンでテンパイするのはかなり攻めてる、普通はしない)。

 字牌 3 - - - -
 １・９ 3 16 - - -
 ２・８ 3 16 16 - -
 ３・７ 3 16 16 16 -
 ４〜６ 3 16 16 - 16

防御を意識しながら手札を作りたい場合、あえて字牌などを序盤で切らずに持っておくという手もある。

スジ

字牌も使い切った場合。

一番強い (相手目線で狙いたい) 待ち方はリャンメン。なのでリャンメンに当たらないとわかるだけで、結構安心感がある。

リャンメンを仮定した時、たとえば相手の場に 4 が捨てられてるなら、 1 はロンされない。なぜ？ 1 をロンするリャンメンは (2, 3) だけど、その場合は 4 も待ちに含まれているはずなので、フリテンになっている。

よく考えると、リャンメンは以下の６通りしかない。

table: リャンメン
 1 2 3 4 5 6 7 8 9
 X - - X - - X
  X - - X - - X
   X - - X - - X

ここからたとえば、「1 と 7 が両方捨てられているなら、 4 はリャンメンに当たらない」こともわかる。あるいは「 1 しか捨てられていない」場合の 4 は、 (２３) には当たらないけど (５６) には当たるのでまだちょっと怖い、とかもわかる。

このような２個飛ばしの組、たとえば (1, 4, 7) をスジという。一般に、スジの両端が捨てられているなら真ん中が、逆に真ん中が捨てられているなら両端が、リャンメンに「絶対」ロンされない牌。

スジが見えている場合、

table: 当たる待ち
  シャンポン (11X) 内側のリャンメン (X23) カンチャン (1X3) ペンチャン (12X) 外側のリャンメン (23X)
 字牌 3 - - - -
 １・９ 3 0 - - -
 ２・８ 3 0 16 - -
 ３・７ 3 0 16 16 -
 ４〜６ 3 0 16 - 0

１・９はもはや単騎/シャンポンしかあり得ないのでかなり安全 (字牌と同じくらい)。それ以外の牌は、比較的安全だがカンチャンの可能性がまだある。

３・７は、カンチャンに加えてペンチャンも残るので、スジが見えていても要注意 ( 1 しか見えてない時の 4 くらいには危険)。

リーチ牌に注目する

相手がリーチするときに出した牌は、リーチの直前まで温存していた牌とも言えるから、おそらくなにかのフォロー牌になっている (孤立牌をリーチ直前まで持っているのは不自然)。したがって、その±３以内の牌は当たる可能性が結構ある。

シャンポンがフォローしていた場合
 例： 相手 (④⑤５５６) で１シャンテン
 1. ③, ⑥を引く → ５を捨てて (５６) のリャンメンでリーチ (４, ７待ち) ... リーチ牌±２以内が当たり
 2. ４, ５, ７を引く → ５か６を捨てて (④⑤) のリャンメンでリーチ (③, ⑥待ち)

リャンカンになっていた場合
 例： 相手 (④⑤２４６) で１シャンテン
 1. ③, ⑥ を引く
   ２を捨てて (４６) のカンチャンでリーチ (５待ち) ... リーチ牌＋３が当たり
   ６を捨てて (２４) のカンチャンでリーチ (３待ち) ... リーチ牌ー３が当たり
 2. ３, ５ を引く → ２か６を捨てて (④⑤) リャンメンでリーチ (③, ⑥待ち)

特にリャンカンのパターンは、ちょうどリーチ牌のスジが当たりになっている (モロヒッカケ) ので油断しやすい。

逆に相手が早々に切っている牌ほど、孤立牌や、ブロックが完成していらなくなったフォロー牌などの可能性が高まるので、その周囲は安全寄りになる。特に相手が早々に切った牌のスジは、リャンメンだけでなくカンチャンにも当たらない可能性が高い。ただし中盤以降に切られた牌のスジは、カンチャン/シャンポン待ちになりそうなことあらかじめ予期して、安全に見せるために意図的にスジを捨ててることもあるので注意。

ノーチャン/ワンチャン、カベ

すでに４枚見えていてもう残っていない牌をノーチャン、１枚しか残っていない牌をワンチャンという。

たとえば東を自分が３枚持っている時、これを捨ててもロンされない。なぜ？東は世の中にあと１枚しかないから、相手は絶対に東でシャンポン待ちできない (単騎待ちに当たる可能性はあるけど、あんまり現実的じゃない)。

たとえば自分が (２３３３) を持っている時、２を捨ててもたぶんロンされない。なぜ？シャンポンに当たるのは仕方ないとして、他に怖いのは (１３) のカンチャンと (３４) のリャンメンだけど、３は世の中にあと１枚しかないから特定の相手が持っている確率は低い。

table: 当たる待ち
  シャンポン (11X) 内側のリャンメン (X23) カンチャン (1X3) ペンチャン (12X) 外側のリャンメン (23X)
 ２ 3 4 4 - -

一般に、３〜４枚がすでに見えている牌 (カベ) の隣 (特に外側) は安全な可能性が高い、と覚えておくと便利。カベの位置が外であるほどより安全。なぜ？カベが内側になっていくにつれ、ペンチャンや外側からのリャンメンを考慮しないといけなくなるため。

残り枚数は微差の判断にも使える。たとえば警戒している相手が⑤, ５を捨てていて、こちらは⑤のスジにあたる②か、５のスジにあたる２を捨てたいとする。このとき、もし別のプレーヤーが１を 2 枚捨てていることに気付ければ、同じスジでも②より２の方がちょっと安全だとわかる。

table: 当たる待ち
  シャンポン (11X) 内側のリャンメン (X23) カンチャン (1X3) ペンチャン (12X) 外側のリャンメン (23X)
 ② 3 0 16 - -
 ２ 3 0 8 - -

ひとグループまるっと捨てている場合

モロヒッカケにあたるスジでも、相手がひとグループまるっと捨てている場合、安全な可能性がある。

たとえば相手が６, ５を捨ててリーチした場合、そのスジにあたる２, ８は安全。なぜ？

 ２について、警戒すべきは (１３) のカンチャン。このとき相手は直前まで (１３５６) を持っていたことになるが、そこで (５６) を切るのは不自然。 (１３) を先に切った方が受け入れが広い。

 ８について、警戒すべきは (７９) のカンチャン。このとき相手は直前まで (５６７９) を持っていたことになるが、それなら９を切って上がっているはず。

ペアをまるっと捨てている場合も同様。

たとえば相手が９, ９, ５を捨ててリーチした場合、そのスジのうち８は比較的安全。なぜ？警戒すべきは (７９) のカンチャン。このとき相手は直前まで (５７９９９) を持っていたことになるが、そこで積極的に９を切るのは不自然。

情報が少ない場合

とりあえず字牌は比較的安全 (待ちパターンが物理的に少ない) なので、字牌があればそれを切る。

もしペアやスリーカードができているなら、優先して切る。特に１・９牌だと嬉しい。当たってしまう可能性はあるが、スルーされた場合それだけで以降２〜３巡しのげるので、賭けに勝った時のリターンが大きい (毎ターン違う牌を捨ていく場合、その中のどれか１つでも当たればアウトなので、結果的に確率は上がってしまう)。
